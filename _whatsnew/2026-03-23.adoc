=== Oracle Database workloads enhancements
In this release, Oracle Database workloads introduces the following enhancement:

*Clustered host support*: You can now discover and protect Oracle Real Application Clusters (RAC) databases that use Oracle Automatic Storage Management (ASM) and are deployed using VMDK on VMFS file systems.

For details about protecting Oracle Database workloads, refer to https://docs.netapp.com/us-en/data-services-backup-recovery/br-use-oracle-protect-overview.html[Protect Oracle Database workloads overview].
