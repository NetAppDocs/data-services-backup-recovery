=== KVM workloads supported in General Availability (GA)
Microsoft Hyper-V workload support is now generally available (GA) in NetApp Backup and Recovery.

For details about protecting KVM workloads, refer to https://docs.netapp.com/us-en/data-services-backup-recovery/br-use-kvm-protect-overview.html[Protect KVM workloads overview].

=== Kubernetes workloads enhancements
Kubernetes workload support is now generally available (GA) in NetApp Backup and Recovery. This release of Kubernetes workloads also introduces the following enhanced capabilities:

* *Reporting support*: You can now view a status snapshot of your protected Kubernetes workloads and generate reports from the Reporting page within the Backup and Recovery web UI.

For details about protecting Kubernetes workloads, refer to https://docs.netapp.com/us-en/data-services-backup-recovery/br-use-kubernetes-protect-overview.html[Protect Kubernetes workloads overview].
